# SRI-Alumbrado-Publico-Mexticacan-MVP
Prototipo MVP de un sistema integral para el reporte, gestión y seguimiento de incidencias en la red de alumbrado público del municipio de Mexticacán, Jalisco.

Descripción del Proyecto
Este sistema permite a los ciudadanos y administradores municipales reportar fallas en luminarias de manera geolocalizada. El objetivo es optimizar los tiempos de respuesta del departamento de servicios públicos y mantener un historial digital de mantenimientos.

Funcionalidades Clave:
-Reporte Ciudadano: Formulario ágil para reportar fallas (foco fundido, poste dañado, etc.).
-Panel Administrativo: Gestión de estados de incidencia (Iniciado, En Proceso, Concluido).

Stack Tecnológico
-Framework: Laravel 12.44.0
-Lenguaje: PHP 8.2.12
-Base de Datos: MySQL
-Frontend: Blade Templates (Bootstrap)

Arquitectura de la Base de Datos
El sistema se basa en las siguientes entidades principales:
Incidencias: Almacena ubicación, tipo de falla, evidencia fotográfica y estado (Iniciado, En Proceso, Concluido).
Usuarios: Distingue entre ciudadanos y administradores.
Zonas/Sectores: División geográfica limitada al territorio del municipio de Mexticacán para mejor organización.
